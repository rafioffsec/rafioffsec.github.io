# [](#Apa-itu-File-History-?)Apa itu File History ?

File History adalah fitur bawaan yang ada di dalam Windows 11 yang memungkinkan pengguna dapat memulihkan file secara otomatis. fitur ini berguna untuk melindungi file pribadi seperti dokumen, foto, dan video agar tidak hilang ataupun rusak.
